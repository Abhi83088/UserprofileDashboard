# User Profile Dashboard

## Description
A responsive and interactive User Profile Dashboard built with HTML, CSS, JavaScript, and Bootstrap. 

## Features
1. Profile Section: Display user details and change profile picture.
2. Edit Profile Feature: Dynamically update user information with validation.
3. Recent Activities: View activities with smooth animations.
4. Settings Section: Toggle dark mode and notifications (preferences saved in localStorage).
5. Mobile Responsiveness: Fully responsive on all devices.
